Le site a été réalisé en HTML et CSS. Le HTML permet de structurer le contenu (header, navigation, sections et footer) tandis que le CSS est utilisé pour la mise en forme graphique. Les couleurs choisies — beige, or et blanc — transmettent une image élégante, sérieuse et rassurante, adaptée au secteur bancaire.

Le projet a été développé par étapes, en respectant une méthodologie en sprints, ce qui a permis d’organiser le travail, d’améliorer progressivement le code et d’obtenir un résultat clair, lisible et professionnel.
